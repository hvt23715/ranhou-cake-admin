## 1. 项目概述

### 1.1 项目背景

- **客户**：燃厚蛋糕（吉林省长春市头部烘焙品牌，约20家直营门店）
- **目标**：结合店大师能力，为燃厚蛋糕设计 B端管理系统
- **定位**：店大师第一次 ToB 尝试，打造烘焙行业数字化管理标杆案例

### 1.2 产品定位

面向烘焙连锁品牌的**智能经营管理平台**，以 AI 问答为核心交互方式，融合地理数据洞察与行业知识库，为老板/管理者提供决策支持。

### 1.3 设计理念

```
┌─────────────────────────────────────────────────────────┐
│                    以 AI 问答为中心                       │
│                                                          │
│   数据查询 ←→ 行业知识 ←→ 地理洞察 ←→ 预警提醒           │
│                                                          │
│                    + 固定数据看板                        │
└─────────────────────────────────────────────────────────┘
```

## 4. 功能模块

### 4.1 AI 问答中心（核心）

**设计理念**：像和业务助手聊天一样管理门店

#### 4.1.1 问答交互界面

- 对话式交互，底部输入框
- 支持文字输入
- AI 回复支持：文字、数据表格、图表、建议卡片

#### 4.1.2 建议问题（快捷入口）

点击即问，无需打字：

| 类别 | 示例问题 |
|------|----------|
| **数据查询** | "今天全部门店的销售额是多少？" |
| | "上周销量最高和最低的门店分别是哪家？" |
| | "本月各品类的销量占比是多少？" |
| **运营建议** | "哪些单品最近3天销量持续下降？" |
| | "帮我分析一下上周的销售数据，有什么问题？" |
| | "根据近期数据，给出下周的备货建议" |
| **行业知识** | "面包店如何提升下午时段的客流？" |
| | "生日蛋糕的库存周转率一般多少正常？" |
| | "烘焙行业最近有什么新趋势？" |
| **违规检测** | "今天有门店出现食品安全违规情况吗？" |
| | "哪些门店的卫生评分下降了？" |
| | "员工操作规范方面有什么问题？" |
| **地理洞察** | "分析一下各门店周边的竞争对手情况" |
| | "哪些门店选址附近的客流密度最高？" |
| | "基于门店分布，给出最优配送路线建议" |

#### 4.1.3 AI 能力融合

结合店大师能力：

| 能力 | 说明 |
|------|------|
| **地理数据问答** | 基于高德地图数据，分析门店周边 POI、客流、竞品分布 |
| **行业知识库** | 烘焙行业的运营知识、最佳实践、常见问题解答 |

### 4.2 固定数据看板

**位置**：首页/仪表盘

#### 4.2.1 核心指标卡片

| 指标 | 说明 |
|------|------|
| 今日销售额 | 当天累计销售额，支持同比 |
| 门店总数 | 当前在营门店数量 |
| 今日订单数 | 当天有效订单总数 |
| 平均客单价 | 当日销售额/订单数 |

#### 4.2.2 固定图表

| 图表 | 类型 | 说明 |
|------|------|------|
| 销量趋势 | 折线图 | 近7天/30天销量走势 |
| 品类销量排行 | 柱状图 | 面包、蛋糕、甜点等品类销量对比 |
| 门店销量排行 | 条形图 | 各门店销量排名 TOP10 |
| 销售时段分布 | 柱状图 | 早/中/晚各时段销售占比 |

---

### 4.3 最新动态预警

**位置**：侧边栏/首页卡片

#### 4.3.1 预警类型

| 类型 | 触发条件 | 展示内容 |
|------|----------|----------|
| 🔥 爆单预警 | 单品1小时内订单量 > 50 | "【爆单】芒果慕斯最近1小时订单量激增150%" |
| ⚠️ 滞销预警 | 单品连续3天销量 < 5 | "【滞销】提拉米苏已连续3天日均销量不足5件" |
| 📈 门店增长 | 单店日销量环比 > 50% | "【增长】红旗街万达店今日销量环比+68%" |
| 📉 门店下滑 | 单店日销量环比 < -30% | "【下滑】欧亚卖场店今日销量环比-42%" |
| 🆕 新品表现 | 上新7天内单品数据 | "【新品】草莓芝士蛋糕上市5天，日均销量32件" |
| 🎯 目标达成 | 门店/品类销售目标进度 | "【目标】面包品类本周完成率78%，还需XX" |

#### 4.3.2 预警展示

- 时间倒序，最新在上
- 支持按类型筛选
- 支持「已处理」标记

---

### 4.4 地图热力展示

**技术**：高德地图 API

#### 4.4.1 门店分布地图

- 标记所有门店位置
- 点击门店标记显示：店名、地址、今日销量、实时客流
- 支持缩放、拖拽

#### 4.4.2 销量热力

- 热力图形式展示各门店销量分布
- 颜色梯度：红（高销量）→ 蓝（低销量）
- 支持时间筛选：今日/本周/本月

#### 4.4.3 客流热力

- 基于门店周边客流数据
- 颜色梯度：红（高客流）→ 蓝（低客流）
- 可对比销量热力与客流热力，分析转化率

#### 4.4.4 地理洞察（结合店大师）

- 门店周边 POI 分析：竞争对手分布、潜在客群密度
- 选址建议：基于已有门店数据，识别潜力区域

---

### 4.5 摄像头实时查看

**说明**：Demo 阶段使用模拟视频流

#### 4.5.1 功能

| 功能 | 说明 |
|------|------|
| 门店选择 | 下拉选择门店 |
| 实时画面 | 显示门店摄像头画面（Demo 用占位图） |
| 抓图 | 一键截取当前画面 |
| 违规检测 | AI 识别违规情况（Demo 显示模拟结果） |

#### 4.5.2 违规类型

| 违规类型 | 说明 |
|----------|------|
| 员工着装 | 未佩戴帽子/口罩 |
| 卫生问题 | 操作台不洁 |
| 安全问题 | 陌生人员进入后厨 |
