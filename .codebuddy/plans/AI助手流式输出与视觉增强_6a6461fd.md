## AI 对话增强

- **流式输出**：在首页 AI 智能助手和 AI 助手详情页实现打字机式的流式响应效果，提升用户交互体验。
- **首页卡片优化**：固定首页 AI 卡片高度，确保页面布局稳定，防止因对话内容增多导致卡片撑开；增加更多经营相关的建议问题（Quick Questions）。

## 监控系统更新

- **画面替换**：将首页监控预览及监控详情页的 3 张 Unsplash 图片替换为真实的蛋糕店/烘焙坊内部场景，增强系统代入感。

## 地图视觉更新

- **长春背景**：将首页门店分布图的背景图替换为长春市城市景观，或通过高德地图 API 实现更精准的地理展示，以符合业务实际所在地。

## 技术方案

- **流式接口**：修改 `src/lib/ai.ts`，利用 `fetch` API 的 `ReadableStream` 特性配合 `TextDecoder` 解析 SSE（Server-Sent Events）数据流。
- **状态管理**：在 `DashboardAIChat` 和 `AIAssistant` 组件中使用异步迭代器更新消息列表，实现内容的增量显示。
- **样式约束**：通过 Tailwind CSS 的 `max-h` 或固定 `height` 类名
名（如 `h-[500px]`）结合 `overflow-y-auto` 确保容器高度受控。
- **静态资源**：更新 `DashboardCameraPreview` 中的常量数组，使用特定的 Unsplash 蛋糕店场景 ID。

## 架构调整

### 系统架构

- **API 层**：新增 `fetchAIResponseStream` 工具函数。
- **UI 层**：修改 `DashboardAIChat`、`AIAssistant` 适配流式状态逻辑；修改 `DashboardCameraPreview` 和 `DashboardMapPreview` 更新静态内容。
