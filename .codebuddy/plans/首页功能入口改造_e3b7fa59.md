## 产品概述

在燃厚蛋糕 B端管理系统首页 Dashboard 中，增加其他功能模块的快捷入口，方便用户快速访问核心功能。

## 核心功能

- 在首页仪表盘添加三个功能入口卡片：AI智能助手、门店地图、摄像头监控
- 每个卡片包含功能图标、名称、简要描述
- 点击卡片或按钮可直接跳转到对应功能页面
- 卡片设计与现有 StatCard 保持一致的视觉风格

## 技术栈

- React 18 + TypeScript
- React Router v6（useNavigate 实现页面跳转）
- Tailwind CSS（现有样式系统）
- lucide-react（图标库）

## 实现方案

### 修改点

在 Dashboard.tsx 的统计数据区域下方，新增功能快捷入口区域：

- 使用 `useNavigate` 进行路由导航
- 复用现有的 `.card` 样式类
- 每个卡片展示对应功能的图标、标题、描述和跳转按钮

### 设计一致性

- 使用项目现有的 primary 配色（#e1584e 红色系）
- 卡片圆角、阴影与 StatCard 保持一致
- 布局采用响应式网格（grid-cols-1 md:grid-cols-3）
